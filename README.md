# HR Analytics Dashboard

## Project Overview
The HR Analytics Dashboard is an interactive Tableau dashboard designed to analyze employee attrition trends and workforce demographics. It helps HR professionals and management understand employee turnover patterns, identify high-risk groups, and make data-driven decisions for workforce planning and retention.

## Tools & Technologies
- Tableau Desktop
- Microsoft Excel
- SQL (MySQL)
- Data Visualization

## Dashboard Features

### KPI Cards
- Total Employee Count
- Attrition Count
- Attrition Rate (%)
- Active Employees
- Average Employee Age

### Visualizations
1. Department-wise Attrition Analysis
2. Employee Distribution by Age Group
3. Attrition by Gender
4. Job Satisfaction Rating Analysis
5. Education Field-wise Attrition
6. Attrition Rate by Gender Across Age Groups

### Filters
- Education Level Filter

## Business Objectives
- Monitor employee attrition trends.
- Identify departments with high turnover.
- Analyze attrition based on age, gender, and education.
- Evaluate employee satisfaction levels.
- Support HR decision-making with data insights.

## Key Insights
- Overall attrition rate is 16.12%.
- Sales department shows the highest attrition.
- Employees aged 25-34 have the highest attrition count.
- Life Sciences and Medical fields contribute significantly to attrition.
- Male employees show higher attrition compared to female employees.
- Job satisfaction impacts employee retention.
